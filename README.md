# Fake News Detection using BERT

## Overview
This project implements a Fake News Detection system using a fine-tuned BERT model for binary text classification.

## Dataset
Used ISOT Fake News Dataset from Kaggle.

## Methodology
- Data preprocessing and labeling
- Tokenization using BERT tokenizer
- Fine-tuning BertForSequenceClassification
- Evaluation using accuracy and F1-score
